Description

The Personal Book Library Website is a web application designed to manage a personal collection of books. It allows users to organize books, search easily, track borrowing activity, and maintain a clean and attractive digital library.

This project was developed as Task 3 during my Web Developer Internship at HexSoftwares.


Technologies Used

- HTML5
- CSS3
- JavaScript



Features

- User registration and login
- Add, edit, and delete books
- Book categorization
- Search books by title or author
- Borrow and return books
- Borrowing history tracking
- Option to clear borrowing history
- Dashboard with book statistics
- Responsive and user-friendly interface
